Frequently Asked Questions (FAQ)

When will credits be deducted from my account?
Credits will be deducted only when you register your face. When you verify your face, credits will not be deducted.
Can the same username be used within the same API token?
You cannot use the same username within the same API token.
When a user deletes a created group, what happens to the users in that group?
When a group is created and later deleted, the users in that group will be moved to the Default User group.
